Basic functions $log({string})

Adds a {string} to the log.

$sleep({int})

Blocks the calling script for {int} milliseconds.

$use({string})

This method is required by some hardware features.

$open({string}, {string})

This method will open a communication channel.

Examples:
WebSocket: {IP}:50505/ws/echo <=> $open("websocket", "echo");
Bluetooth: First pair the device and then you can connect through mac or device name <=> $open("bluetooth", "mac" || "name");

{int}|{string} = $value.get({string})

Returns the corresponding global variable.

$value.set({string}, {int}|{string})

Add/set a global (can be access from any running script) object with a {string} based identificator.

$value.delete({string})

Deletes the corresponding global variable.

Hardware {object} = $accelerometer.read()
Before using call $use("accelerometer")

Returns a {object} which contains the device acceleration.

{object}.x [m/s^2]
{object}.y [m/s^2]
{object}.z [m/s^2]

{object} = $gyroscope.read()
Before using call $use("gyroscope")

Returns a {object} which contains the rotation per second.

{object}.x [rad/s]
{object}.y [rad/s]
{object}.z [rad/s]

 Communication {string} = $bluetooth.read()
* see $open("bluetooth", {string})

Waits until a message appears and return the received message as a {string}.

$bluetooth.write({string})
* see $open("bluetooth", {string})

Sends the {string} to the target device.

$websocket.waitForClient()
Before using call $open("websocket", "endpointName")

This method will block the calling script until a client is connected.

{string} = $websocket.read()
Before using call $open("websocket", "endpointName")

This method will block the calling script until a message was received.

$websocket.write({string})
Before using call $open("websocket", "endpointName")

Sends a message to the connected client.
