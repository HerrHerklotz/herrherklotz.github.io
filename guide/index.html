Description
aBrain is the first rapid prototyping environment for Android devices. Through a scripting interface you can directly start coding with the included hardware and software features.

My primary development target is to keep it simple for you! Things like speed and stability are only secondary and will be considered later (if I could work full time on the project ;-) ).

To enable a easy development workflow aBrain includes two tools:
  1. Editor: create your scripts
  2. Monitor: supervise the execution
Both are designed as platform independent web app and can be used from any device over network.

http://:8080/editor

http://:8080/monitor

Recommendations
  • use always the latest version of Google Chrome
  • if you use the web application directly from you aBrain running device, you can connect via localhost:8080/.. (even if your device is offline)
  • for connection over mobile network you will need a VPN solution
